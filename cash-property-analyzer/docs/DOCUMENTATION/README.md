# CASH-ONLY PROPERTY ANALYZER

**Doctrine-driven capital allocation for South African residential property**

## 🎯 PURPOSE

This PWA evaluates 100% cash-only residential multi‑let property deals using deterministic scoring based on observable owner earnings. It replaces emotion with encoded discipline.

## 🚨 CRITICAL WARNINGS

1. **Doctrine is immutable** - No bypassing auto-reject gates
2. **Golden references never change** - If they fail, fix the bug
3. **Scoring formula is locked** - Changes require CIO approval
4. **Business logic stays in /core** - UI only displays
